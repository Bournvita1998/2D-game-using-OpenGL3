Controls::

Right Arrow 	->	 	move ball in right direction
Left Arrow 		-> 		move ball in left direction
Space bar 		->		move ball in upward direction

Panning			-> 		Standard W,S,A,D keys
Scroll up 		->		Zoom in
Scroll down		->		Zoom out

Player can control ball using mouse.

Score Displayed in top right corner.
Level been displayed in center top but hardness in game increases as the scores increases.
